### 1. FourPillarsCard.tsx
```typescript

  fontSize: 32,  // 从 24 → 32
  fontWeight: fontWeights.bold,
},
charSmall: {
  fontSize: 24,  // 从 20 → 24
  fontWeight: fontWeights.semibold,
},
```

```typescript

  flexDirection: 'row',
```
