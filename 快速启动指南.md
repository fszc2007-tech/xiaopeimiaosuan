## 📱 连接设备

### Android 模拟器

1. **确保 Android Studio 已安装并创建了 AVD**
   ```bash
   # 检查 Android Studio 是否安装
   ls /Applications/Android\ Studio.app
   
   # 检查是否有 AVD
   emulator -list-avds
   ```

2. **启动模拟器**（两种方式）：
   ```bash
   # 方式 1: 命令行启动
   emulator -avd <你的AVD名称>
   
   # 方式 2: 在 Android Studio 中点击播放按钮
   ```

3. **在 Expo 终端中按 `a` 键**，Expo 会自动连接

### iOS 模拟器

1. **确保 Xcode 已安装**
2. **在 Expo 终端中按 `i` 键**，会自动启动 iOS 模拟器

### 真机测试

1. **下载 Expo Go App**（iOS App Store / Google Play）
2. **扫描 Expo 终端中显示的二维码**
3. App 会自动加载

## ⚠️ 常见问题

### Q: 提示 "cd: no such file or directory: app"

**A**: 你不在项目根目录。请先运行：
```bash
cd /Users/gaoxuxu/Desktop/xiaopei-app
```

### Q: 提示 "Need to install the following packages: expo"

**A**: 这是正常的，输入 `y` 确认安装即可。

### Q: Android 模拟器无法连接

**A**: 
1. 确保模拟器已启动
2. 检查 ADB 连接：
   ```bash
   adb devices
   ```
3. 如果显示 "unauthorized"，在模拟器上点击 "Allow"
