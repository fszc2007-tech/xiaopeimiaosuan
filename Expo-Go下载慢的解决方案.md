# Expo Go 下载慢的解决方案

## 🚀 解决方案（3 种方法）

### 方法 1: 耐心等待（最简单）

- ✅ **优点**: 不需要任何操作
- ⏳ **时间**: 通常需要 5-15 分钟（取决于网速）
- 📊 **进度**: 当前 8%，继续等待即可

**建议**: 如果网络正常，继续等待是最简单的。

---

### 方法 2: 手动安装（可能更快）

如果下载太慢，可以手动安装：

#### 步骤 1: 取消当前下载

在 Expo 终端中：
- 按 `Ctrl + C` 取消当前下载

#### 步骤 2: 在模拟器中手动安装

1. **打开 Play Store**
   - 在模拟器中点击 Play Store 图标

2. **搜索 "Expo Go"**
   - 在 Play Store 中搜索
   - 找到 Expo Go（由 Expo 开发）

3. **安装**
   - 点击 "Install"
   - Play Store 的下载通常比 adb 安装更快

4. **连接**
   - 打开 Expo Go
   - 输入: `exp://localhost:8081`
   - 点击 "Connect"

---

### 方法 3: 使用预下载的 APK（最快）

如果网络真的很慢，可以手动下载 APK：

#### 步骤 1: 下载 Expo Go APK

访问 Expo Go 的下载页面：
- 或者使用命令：
```bash
# 下载 Expo Go APK（最新版本）
curl -L -o /tmp/expo-go.apk "https://d1a1f0psnxq1cc.cloudfront.net/Exponent-2.28.10.apk"
```

#### 步骤 2: 安装 APK

```bash
# 安装到模拟器
adb install /tmp/expo-go.apk
```

#### 步骤 3: 连接

- 打开 Expo Go
- 输入: `exp://localhost:8081`
- 点击 "Connect"

## 🎯 推荐操作

### 如果下载速度还可以（> 1 Mbps）

## ⚠️ 注意事项

- **不要中断**: 如果下载进行中，尽量不要中断
- **网络稳定**: 确保网络连接稳定
