## 🎨 设计理念

让每个五行的颜色**符合其属性特征**，使用户一眼就能识别并感受到五行的本质。

## 🎯 设计原则

### 1. 符合传统五行理论
```
木 → 东方 → 青色 → 春天 → 生长
火 → 南方 → 赤色 → 夏天 → 炎热
土 → 中央 → 黄色 → 长夏 → 化育
金 → 西方 → 白色 → 秋天 → 收敛
水 → 北方 → 黑色 → 冬天 → 寒藏
```

### 2. 现代化视觉呈现
- 保留传统五色：青、赤、黄、白、黑
- 调整为现代审美的色调
- 提高饱和度和辨识度

### 3. 和谐的整体搭配
- 5 种颜色相互协调
- 前景色与背景色成套
- 视觉层次分明
