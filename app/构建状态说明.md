## 构建进度检查

### 方法 1: 使用自动脚本
```bash
cd app
npm run auto:ios:skip-build  # 只检查状态，不构建
```

### 方法 2: 手动检查
```bash
# 检查构建目录
ls -la ios/build/

# 检查构建产物
find ios/build -name "*.app" -type d

# 检查构建进程
ps aux | grep xcodebuild
```

## 监控构建

### 查看实时日志
```bash
# 如果构建在后台运行，可以查看日志
tail -f ~/Library/Logs/CoreSimulator/*.log
```

### 检查 CocoaPods 安装
```bash
cd app/ios
pod install --repo-update
```

## 常见问题

### 1. 构建卡住
- 检查 Xcode 是否已安装
- 检查 Command Line Tools 是否已安装
- 尝试清理后重新构建

### 2. CocoaPods 安装失败
```bash
cd app/ios
pod deintegrate
pod install
```

### 3. 构建超时
- 首次构建可能需要更长时间
- 检查网络连接（需要下载依赖）
- 检查磁盘空间
