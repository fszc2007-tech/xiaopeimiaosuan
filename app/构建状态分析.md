## 🧹 清理多余的 expo start 进程（可选）

如果觉得资源占用过多，可以关闭多余的进程：

```bash
# 查看所有 expo start 进程
ps aux | grep "expo start" | grep -v grep

# 关闭特定进程（替换 PID）
kill <PID>

# 或者关闭所有 expo start（谨慎使用）
pkill -f "expo start"
```

**注意**: 关闭 `expo start` 不会影响 `xcodebuild` 的构建过程。
