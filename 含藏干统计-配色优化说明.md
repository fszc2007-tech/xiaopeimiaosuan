# 含藏干统计卡片 - 配色优化说明

## 🎨 配色调整

### 修改内容

将卡片背景色和线框颜色从**金黄色系**改为**系统绿色系**，与 App 主品牌色保持一致。

---

## 📊 具体修改

### 1. 表头背景色
- ❌ 旧：`#d8b574`（金黄色）
- ✅ 新：`colors.brandGreen` = `#52b788`（品牌绿）

### 2. 卡片边框
- ❌ 旧：`#f3ebda`（浅土色）
- ✅ 新：`colors.brandGreen` = `#52b788`（品牌绿）

### 3. 柱子分隔线
- ❌ 旧：`#f2e9da`（浅土色）
- ✅ 新：`colors.greenSoftBg` = `#d8f3dc`（浅绿色）

### 4. 条形图轨道背景
- ❌ 旧：`#f7f4ed`（浅米色）
- ✅ 新：`colors.greenSoftBg` = `#d8f3dc`（浅绿色）

### 5. 五行颜色（保持不变）✅
- 木：`#52b788`
- 火：`#ff6b6b`
- 土：`#d4a373`
- 金：`#ffd700`
- 水：`#4a90e2`

---

## ✅ 优势

1. **品牌一致性**：与 App 主色调（绿色系）保持一致
2. **视觉和谐**：整体色调统一，更加协调
3. **用户体验**：符合用户对系统配色的预期
4. **五行完整**：五行颜色保持原有专业配色

---

## 📝 系统配色规范

根据 `app/src/theme/colors.ts`：

```typescript
// 品牌主色 - 绿色系
primary: '#52b788',         // 主绿色：主按钮、重点操作、品牌主色
primaryPressed: '#2d6a4f',  // 主色按下态：深绿
primaryLight: '#95d5b2',    // 浅主色：轻提示、次要按钮

brandGreen: '#52b788',      // 主绿色（兼容旧代码）
greenSoftBg: '#d8f3dc',     // 浅绿底，轻提示、示例 chip
```
