## 📱 模拟器启动流程

### 1. 等待模拟器完全启动

模拟器需要 **1-2 分钟** 才能完全启动。你可以：

- **查看模拟器窗口**：会显示 Android 启动动画
- **等待主屏幕出现**：看到 Android 主界面表示启动完成

### 2. 检查连接状态

运行检查脚本：

```bash
cd /Users/gaoxuxu/Desktop/xiaopei-app
./scripts/check-emulator-status.sh
```

或者手动检查：

```bash
adb devices
```

应该显示：
```
List of devices attached
emulator-5554    device
```

### 3. 启动 Expo 开发服务器

模拟器启动完成后：

```bash
cd /Users/gaoxuxu/Desktop/xiaopei-app
./start-expo.sh
```

### 4. 连接模拟器

在 Expo 终端中：
- 按 **`a`** 键连接 Android 模拟器
- App 会自动安装并启动

## ⚠️ 常见问题

### Q: 模拟器启动很慢

**A**: 正常，首次启动需要 1-2 分钟。后续启动会快一些。

### Q: adb devices 显示 "unauthorized"

**A**: 在模拟器窗口上点击 "Allow" 授权。

### Q: Expo 无法连接

**A**: 
1. 确保模拟器完全启动（看到主屏幕）
2. 运行 `adb devices` 确认设备已连接
3. 重启 Expo：`npx expo start --clear`
