# 安全策略文档目录

本目录包含所有与系统安全相关的文档。

## 文档列表

- `Core架构与安全策略方案.md` - 总体架构和安全策略方案
- `前端开发安全规范.md` - 前端开发必须遵守的安全规范（**系统级要求**）
- `后端开发安全规范.md` - 后端开发必须遵守的安全规范（**系统级要求**）
- `代码审查安全检查清单.md` - 代码审查时使用的安全检查清单

## 核心原则

1. **前端只负责 UI，不包含核心逻辑**
2. **所有算法、prompt、计费判断都在 Core 后端**
3. **通过 API 通信，前端无法获取核心资产**
4. **系统化规范，确保持续遵循**
